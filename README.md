# DebugMate AI 🚀

An AI-powered debugging assistant that analyzes code and error messages, explains issues in plain English, and suggests fixes with corrected code.

## Features
- Multi-language support (JS, Python, C++, Java)
- AI-powered explanations (Gemini)
- Clean UI with real-time results
- Copyable corrected code

## Tech Stack
- Next.js
- Tailwind CSS
- Gemini API
- Vercel (deployment)
